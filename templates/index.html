Encryption

Press the "Browse..." button below to select an image you want to encrypt, then press "Encrypt". Your download should start shortly after.

After encrypting, copy the secret key below and store it in a secure place.


Decryption

Press the "Browse..." button below to select an image you want to decrypt, paste the secret key obtained when encrypting the image into the text box on the right, then press "Decrypt". Your download should start shortly after.
